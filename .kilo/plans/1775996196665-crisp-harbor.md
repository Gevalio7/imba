# План: Добавление полей шаблонов к очереди

## Цель
Добавить выбор 5 шаблонов из TemplateQueues к каждой очереди в файле `QueuesAddEdit.vue`:
- шаблон открытия обращения
- шаблон закрытия обращения
- шаблон подтверждения обращения
- шаблон изменения статуса
- шаблон получения комментария по заявке

## Файл для изменения

### `/src/pages/apps/QueuesAddEdit.vue`

## Реализация

### Шаг 1: Обновить интерфейс Queue (строка 5-28)
Добавить 5 новых полей:
```typescript
templateOpenTicketId: number | null      // шаблон открытия обращения
templateCloseTicketId: number | null    // шаблон закрытия обращения
templateConfirmTicketId: number | null  // шаблон подтверждения обращения
templateStatusChangeId: number | null   // шаблон изменения статуса
templateCommentTicketId: number | null  // шаблон получения комментария
```

### Шаг 2: Обновить объект queue (строка 86-109)
Добавить новые поля со значением null в объект queue.

### Шаг 3: Добавить загрузку шаблонов (referenceData)
- Добавить поле templates в ReferenceData
- Добавить fetch для загрузки шаблонов из `/templates`

### Шаг 4: Добавить computed для options шаблонов
Создать computed templateOptions для выбора шаблонов.

### Шаг 5: Обновить saveQueue (строка ~285)
Добавить передачу новых полей в queueData при сохранении.

### Шаг 6: Добавить поля в форму (строка ~360-572)
Добавить 5 полей AppSelect после существующих полей:
- После "Почтовый аккаунт" (postMasterMailAccountId)
- Каждое поле использует templateOptions
- Все поля clearable

## Примечание
- Шаблоны берутся из API `/templates`
- Типы задавать не надо - типы заполняются в очереди отдельно
- Поля используют тот же справочник TemplateQueues
