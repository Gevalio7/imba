## Этап 5: Рефакторинг `TicketEdit.vue`

После рефакторинга:

```vue
<script setup lang="ts">
import { useTicketForm } from '@/composables/useTicketForm'
import { useTicketWorkflow } from '@/composables/useTicketWorkflow'
import { useTicketComments } from '@/composables/useTicketComments'
import { useTicketAttachments } from '@/composables/useTicketAttachments'
import { useReferenceData } from '@/composables/useReferenceData'

const route = useRoute()
const ticketId = computed(() => Number(route.query.id) || null)

const { ticket, description, save, fetchTicket } = useTicketForm(ticketId)
const { currentWorkflow, availableStatuses, fetchWorkflow } = useTicketWorkflow(() => ticket.typeId)
const { comments, newComment, fetchComments, addComment } = useTicketComments(ticketId)
const { existingAttachments, newAttachments, fetchAttachments } = useTicketAttachments(ticketId)
const { data: refData } = useReferenceData()

onMounted(async () => {
  await Promise.all([fetchTicket(), fetchComments(), fetchAttachments()])
})
</script>

<template>
  <div>
    <TicketEditHeader @save="save" @cancel="router.push('/apps/tickets')" />
    
    <VRow>
      <VCol cols="12" md="8">
        <TicketMainInfo v-model:title="ticket.title" v-model:description="description" />
        <TicketAttachments :existing="existingAttachments" :new="newAttachments" />
        <TicketCommentsSection :comments="comments" :new-comment="newComment" @add="addComment" />
      </VCol>
      
      <VCol cols="12" md="4">
        <TicketProperties v-model:ticket="ticket" :reference-data="refData" :statuses="availableStatuses" />
      </VCol>
    </VRow>
  </div>
</template>
```

**Ожидаемый размер**: ~150-200 строк

## Приоритеты внедрения

### Высокий приоритет
1. `src/types/ticket.ts` — типизация
2. `src/utils/slaFormatter.ts` — SLA логика (дублируется)
3. `src/utils/fileUtils.ts` — файловые утилиты
4. `useTicketForm.ts` — форма + сохранение
5. `useTicketComments.ts` — комментарии (сложная логика)

### Средний приоритет
6. `useTicketAttachments.ts` — вложения
7. `useImagePreview.ts` — превью изображений
8. `TicketProperties.vue` — правая колонка
9. `useAuthorSearch.ts` — поиск автора

### Низкий приоритет
10. Полная декомпозиция шаблона
11. Оставшиеся composables
