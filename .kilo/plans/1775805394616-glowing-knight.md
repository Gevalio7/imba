# План: Клонирование тикета по расписанию с префиксом

## Цель
По расписанию клонировать указанный тикет с префиксом "Расписание (Р) " в названии.

## Текущее поведение
- При выполнении расписания создаётся новый тикет на основе сохранённых в расписании полей (`runSchedule` в `ticketSchedulesController.js:145`)
- Название тикета берётся из сохранённого `schedule.title`

## Нужное поведение
- При выполнении расписания клонировать актуальный тикет (источник)
- Добавлять префикс "Расписание (Р) " к названию
